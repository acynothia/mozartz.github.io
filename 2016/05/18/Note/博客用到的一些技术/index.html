博客用到的一些技术

  • note

Gitpage

　　gitpage 是 github 提供的免费空间, 上限是300M. 所有的东西都在自己的 github 上不用害怕丢失. 安全, 方便, 免费.

Domain　　

　　在阿里云上买的域名, 7块钱买了一年. 然后在万网上把域名 CNAME 到 gitpage 分配的域名上, 还要在 github 域名项目的根目录加上 CNAME 文件, 内容是你的域名.

Markdown　　

　　Markdown 的编辑器我用的是作业部落在线编辑的, 挺方便的. 如果觉得这个 Markdown 主题不错还可以抄过来.

Hexo

　　Hexo 是一个简单、快速、强大的博客发布工具，支持Markdown格式。根据对应的主题可以生成漂亮的静态网页了.
　　我用的主题叫 yilia. 其实所有的页面生成都是基于主题的, 所以你也可以将下载好的主题再进行二次修改, 原理类似于用 php smarty 之类的模板引擎, 只不过 Hexo 采用的是 Node.js.

我可以用它来干嘛 ?

  • 整理知识，学习笔记
  • 发布日记，杂文，所见所想
  • 发布技术文章
  • 装 X
